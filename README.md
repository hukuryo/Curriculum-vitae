# (副業)Ambient Lab株式会社(2023年11月〜現在)
## 登録番号兼適格請求書番号管理システム(2024年1月~現在)

## プロジェクト概要
　国税庁が公開している企業の「法人番号」「適格請求書番号」に紐づくデータ(企業名など)を管理し、Web上で番号を確認できるようにするためのシステムの開発。(現在も参画中)
主に番号の検索や、マスタデータ(登録番号、適格請求書番号)の更新などを行える機能が実装されている。

## 担当フェーズ

エンジニア×3名

- アジャイル

## 課題
従来は登録番号や、適格請求書番号を確認する際に、国税庁の公式ページからCSVをダウンロードし、そのCSVファイルの中から該当する番号を探す必要があり、生産性に課題があった。

## 実績・取り組み
- **番号検索機能実装**
DynamoDBに登録された番号を検索できるようにするため、番号を指定して検索する個別番号検索と、CSVファイルをアップロードし、ファイル内に記載されている番号と一致するデータを全て取得する一括検索機能を実装した。

- **マスタデータ更新機能実装**
マスタデータの更新を簡単に行えるようにするため、国税庁のホームページから取得したCSVファイルをアップロードし、そのデータと一致するデータを全件更新する機能と、S3に登録されたデータを比較し、差分のみを更新する機能を実装した。

## 工夫した点
- 検索機能について、データ取得時間を短くするため、AWS OpenSearchを使用することで、高速で検索を行えるようにした。
- 開発効率の向上のため、AmplifyでGitHubへのコミット時に自動デプロイされるように設定を行い、CI/CDを実現した。
- Cognitoを駆使することで、認証機能の実装を簡素化し、開発、運用コストの削減を実現することができた。
